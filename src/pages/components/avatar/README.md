# avatar组件使用文档

> Wepy 小程序组件 - 头像

## API

### Avatar Props

| 名称                  | 描述                         |
|----------------------|------------------------------|
|`src`           |  头像图片地址  |
|`mold`         | 头像规格，circle(正圆)、square(正方)，默认 circle |
|`count` | 消息数，显示在头像右上角 |
|`size` | 头像大小，Number类型，单位rpx |
